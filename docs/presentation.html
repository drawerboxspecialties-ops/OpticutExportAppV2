What it does

Runs in your browser — drop an Allmoxy CSV, get material/edge/ship-date batches, printable cut lists, OptiCut-ready exports, and a live Station queue for the floor.

Import columns

Exported (9): OrderNumber, MaterialName, PartName, W, Length, Quantity, Label, Width, TopEdge.

Batching only: GroupID, Laser, Scoop, Slope, DrillFront, DividersFB, DividersSS, FileSlots, Ship Date.

Batch rules
  • Split by material category + top edge + ship date (optional combine dates)
  • B-edge priority: front inherits matching back’s top edge so foil/raw don’t split
  • Special batches: SPECIAL_ + material + top edge + ship date (never mixed with normal)
  • If any row has Scoop/Slope/Dividers/Drill/FileSlots ≠ None → whole sales order is special
  • Not SPECIAL triggers: GroupID, Laser
  • Name: CAT_EDGE_firstOrder (e.g. PLY_PVC_602480)
  • Each order in exactly one batch; max-orders split available
Print & index
  • Cut list: landscape, 3-col flow; order header 3 boxes (1-2, 2-1); Grp, W, F/B, L/R, Bx, Pcs
  • Code 128 barcode on cut-list header
  • Batch index: all batches + barcodes; orders show group qty 602480 (1-2, 2-1)
  • ★ special; *DFM when front material differs
Station (#station)
  • Header Station link, or open #station
  • Prep: Send to station / Send all
  • Scan barcode or tap batch; checkboxes sync live
  • Remove = soft hide + Add back (amber row)
  • Wipe database = permanent delete all — password dbs
Export & math

Boxes = ceil(parts÷4) per GroupID, then sum for order/batch.

Export strips batching columns. Round widths ON: merges rows, notes originals in Label.

Sidebar ▸: #order · bx; − remove / + add back.

Operator checklist

  1. Export CSV from Allmoxy (include Ship Date + extra columns if used).
  2. Open app → drag & drop → review batches (SPECIAL badge, ship date in meta).
  3. Expand batch ▸ to verify/remove/add orders.
  4. Print Cut list(s) + Batch index → Send to station → export CSV/ZIP → OptiCut.
  5. Floor: open Station, scan barcode, check lines. Wipe all only with password dbs.
