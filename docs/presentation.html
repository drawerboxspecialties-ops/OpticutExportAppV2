DBS Drawers — OptiCut CSV Splitter

One-page guide · Allmoxy CSV → OptiCut cut lists

drawerboxspecialties-ops.github.io/OpticutExportAppV2/

What it does

Runs in your browser — drop an Allmoxy CSV, get material/edge/ship-date batches, printable cut lists, and OptiCut-ready export files.

Import columns

Exported (9): OrderNumber, MaterialName, PartName, W, Length, Quantity, Label, Width, TopEdge.

Batching only: GroupID, Laser, Scoop, Slope, DrillFront, DividersFB, DividersSS, FileSlots, Ship Date.

Batch rules
  • Split by material category + top edge + ship date
  • Special batches: SPECIAL_ + material + top edge + ship date (never mixed with normal)
  • If any row has Scoop/Slope/Dividers/Drill/FileSlots ≠ None → whole sales order is special
  • Not SPECIAL triggers: GroupID, Laser
  • Cut-list table shows full import (Scoop, GroupID, etc.); export strips batching columns
  • Each order in exactly one batch; max-orders split available
Batch sidebar ▸ panel

Click ▸ on any batch to expand orders. Each row shows #order · bx. Use − to remove from batch; click +#order chips to add back.

Cut-list print
  • Landscape letter; two columns of order blocks when space allows
  • Order header once per block with box summary (e.g. 3 boxes (1-2, 2-1))
  • Columns: Grp, W, F/B, L/R, Bx, Pcs, checkbox
  • Identical lines merge; sorted order → width ↓ → length ↓
  • ★ on special groups; batch header lists total boxes + order numbers
Export

Strips all batching columns. Round widths ON: merges rows, blanks W, notes original widths in Label when changed.

Operator checklist

  1. Export CSV from Allmoxy (include Ship Date + extra columns if used).
  2. Open app → drag & drop file → review batches (SPECIAL badge, ship date in meta).
  3. Expand batch ▸ to verify/remove/add orders.
  4. Print Cut list or Cut lists → export CSV or ZIP → import into OptiCut.
